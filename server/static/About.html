Dealership Portal

About Our Dealership Platform

Our Dealership Portal is a full-stack web application designed to help users explore car dealerships, view detailed information about available cars, and read authentic customer reviews. The platform integrates Django, Node.js, MongoDB, and React to deliver a seamless user experience. Users can log in, browse dealerships by state, view dealer-specific reviews, and submit their own feedback with real-time sentiment analysis.
